# ProKit - Project Management App

## Project Overview

ProKit is a React Expo application designed to help users manage their projects effectively. Users can create new projects, add preferred tools, and quickly access recent projects. The app supports both light and dark modes for an enhanced user experience.

## Project Goals

The primary goal of ProKit is to provide a centralized platform for users to organize and manage their projects. This includes:

* **Project Creation:** Allow users to easily create new projects.
* **Tool Integration:** Enable users to add their preferred tools to projects.
* **Quick Access:** Provide shortcuts for accessing recent projects.
* **User Authentication:** Implement secure login, registration, and password reset functionalities.
* **Dark/Light Mode Support:** Offer both light and dark themes for user preference.
